# Reference Architectures

Proven patterns for building applications on the Telnyx Edge platform.

Reference architectures demonstrate how Edge Compute products work together to solve real-world problems. Each architecture includes a diagram, component breakdown, and sample implementation.

## Architectures

  - [Global API Gateway](global-api-gateway.md) — Route API traffic globally with caching, auth, and rate limiting.

  - [Telecom Event Processor](telecom-event-processor.md) — Process voice, SMS, and messaging events in real-time.

  - [Real-Time Media Pipeline](real-time-media-pipeline.md) — Transform and route media streams at the edge.

  - [IoT Data Ingestion](iot-data-ingestion.md) — Collect and process IoT sensor data globally.
