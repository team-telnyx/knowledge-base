# Overview

STT, TTS, and LLM plugin configuration for Telnyx LiveKit agents.

All models are accessed through the [Telnyx plugin](build.md). Models run on Telnyx GPUs, so whether you're using the plugin from LiveKit Cloud or deployed on the Telnyx platform, you get on-prem inference without managing infrastructure.

  - [Speech-to-Text](speech-to-text-stt.md) — Deepgram Nova-3, Nova-2, and Flux — hosted on Telnyx.

  - [Text-to-Speech](text-to-speech-tts.md) — Extensive voice library across multiple providers and models.

  - [LLM](large-language-models-llm.md) — Hosted open-source and proprietary models via BYOK.
